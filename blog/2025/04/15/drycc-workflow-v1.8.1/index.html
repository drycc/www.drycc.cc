Drycc Workflow v1.8.1

New Drycc Workflow Release v1.8.1
15.04.2025

  8 minute read  

!IMPORTANT

This version is a major update with the following major changes made:

  • Remove timeseries
  • Remove telegraf
  • Remove rabbitmq
  • Replace seaweedfs with minio
  • Replace redis with valkey
